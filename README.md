## 🔧 Installation  

Get started with **JSandbox** in just a few steps:  

1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/jsandbox.git
   cd jsandbox
   ```  

2. Install dependencies:  
   ```bash
   npm install
   ```  

3. Launch the application:  
   ```bash
   npm start
   ```
